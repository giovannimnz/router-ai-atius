| Boundary | Description |
|----------|-------------|
| host filesystem → cleanup gate | Paths locais potencialmente destrutivos entram na lista de limpeza. |
| Vault/runtime → Kubernetes Secret | Valores sensiveis cruzam para o cluster sem passar por Git ou evidencia. |
| manifests → k3s scheduler | Configuracao determina node, storage, rede e consumo de recursos. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-29-01 | Tampering | limpeza de disco | high | mitigate | allowlist literal, realpath/owner, inventario before/after e bloqueio de prune/volumes/backups |
| T-29-02 | Information Disclosure | Secret/Vault | high | mitigate | validar nomes de chaves sem ler valores; proibir set -x e payloads em evidencia |
| T-29-03 | Denial of Service | scheduler/PVC | high | mitigate | gate de DiskPressure, affinity obrigatoria e local-path antes do apply |
| T-29-04 | Tampering | imagem do router | high | mitigate | digest arm64 imutavel e identidade registrada sem latest flutuante |
| T-29-SC | Tampering | supply chain de imagens | high | mitigate | usar somente imagens existentes/auditadas e pinadas por digest; nenhuma instalacao de package |
