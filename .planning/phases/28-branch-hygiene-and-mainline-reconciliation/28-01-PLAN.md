<objective>
Create a complete safety envelope before any destructive hygiene action.

This plan inventories all worktrees/branches, captures working state, and writes reversible backup references so later cleanup can proceed without ambiguity.
</objective>

## Deliverables

- immutable inventory of local worktrees and branch/ref mapping
- snapshot of `git status` per worktree
- patch bundles for dirty worktrees
- backup refs/tags for every local worktree HEAD that may be touched

## Tasks

1. Enumerate local worktrees, local branches, upstream tracking, ahead/behind counts, and branch purposes.
2. Capture `git status --short --branch` for each relevant worktree into dated backup artifacts.
3. Generate patch files for every dirty worktree.
4. Create backup refs/tags for each local worktree HEAD before any branch/worktree deletion or reset.

## Acceptance

- no destructive step is allowed until every active worktree has status snapshot + patch + backup ref/tag
- the backup naming scheme is deterministic and easy to correlate to each worktree

## Verification

- `git worktree list --porcelain`
- `git branch -vv`
- `git tag --list 'backup/branch-hygiene-*'`
- verify backup files exist under a dated repo-local backup directory
